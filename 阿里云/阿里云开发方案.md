# 阿里云物联网开发

1. 两种方案

   方案一	阿里云物联网平台+阿里云服务器部署应用接口

   ​		服务器费用 + 0.7344元 每年年每台设备 + 0.2元 每次固件升级

   ​		开发难度和工作量小 稳定性高

   方案二	阿里云服务器搭建mqtt服务和应用接口

   ​		服务器费用

   ​		开发难度和工作量大 稳定性低，但可以分布式部署，可针对不同地区部署服务器，降低		不同地区之间的网络问题
